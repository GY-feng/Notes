# 序列化：
序列化是将对象转换为字节流或其他可存储或传输的格式的过程，而反序列化则是将字节流或其他格式转换回对象的过程。

pickle模块实现序列化：
```python
# pickle模块实现序列化：
import pickle
l=[1,2,3,4,5,6,67]
with open('l.pickle','rb') as f:
    pickle.dump(l,f)#保存
    l=pickle.load(f)#使用
```
json模块实现序列化：
```python
# json模块实现序列化：
import json
l=[1,2,3,4,5,6,67]
with open('l.json','r',encoding='utf-8') as f:
    l=json.load(f)
```
